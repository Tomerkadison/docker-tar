# Docker Tar - Project Brief

## Overview
Docker Tar is a web application that allows users to download Docker images as tar files online. The application provides a user-friendly interface for searching and downloading Docker images.

## Core Requirements
- Search functionality for Docker images
- Download Docker images as tar files
- Modern, responsive web interface
- Fast performance and good user experience

## Technology Stack
- **Frontend Framework**: React 18
- **Build Tool**: Vite (migrated from Create React App)
- **Styling**: Tailwind CSS, Bootstrap, Ant Design
- **UI Components**: React Bootstrap, React Select, Turnstone autocomplete
- **Analytics**: Google Analytics integration

## Key Features
- Autocomplete search with Turnstone
- Tag selection functionality
- Recent searches capability
- Responsive design with custom styling
- Docker-themed branding and icons

## Project Structure
- `src/` - React application source code
- `public/` - Static assets (icons, manifest, etc.)
- `src/components/` - Reusable React components
